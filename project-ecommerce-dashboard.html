E-commerce Sales Analysis Dashboard

This project analyzes sales data from a retail dataset to identify revenue trends, top-performing products, and regional performance. An interactive Power BI dashboard was developed to visualize business insights and help support data-driven decision making.

Problem

Businesses generate large amounts of transactional data, but raw data alone does not easily reveal useful insights. Without proper analysis and visualization, it becomes difficult to identify sales trends, profitable products, and regional performance differences.

The goal of this project was to perform exploratory data analysis on a retail dataset and build an interactive dashboard that allows stakeholders to quickly understand key business metrics.

Dataset

The analysis uses the Kaggle Superstore dataset, which contains information about product sales, profit, order details, customer segments, and regional sales performance.

Tools Used

  • Power BI
  • Data Cleaning
  • Exploratory Data Analysis
  • Data Visualization

Key Insights

The dashboard revealed that certain product categories consistently generated higher revenue while some regions contributed more significantly to total sales.

Visualization of revenue trends helped identify top-performing products and profitable markets, allowing businesses to better understand where growth opportunities exist.
