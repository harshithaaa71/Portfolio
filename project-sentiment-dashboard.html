Sentiment Analysis Dashboard

This project presents an interactive sentiment analysis dashboard built using Python and Plotly Dash. The dashboard analyzes over 500 tweets from the Sentiment140 dataset using VADER sentiment analysis to classify tweets into positive, negative, and neutral categories and visualize sentiment patterns through interactive charts.

Problem

Understanding public opinion from large volumes of textual data is difficult without proper analytical tools. Social media platforms such as Twitter contain valuable insights about public sentiment, but raw text data must be processed and analyzed to extract meaningful patterns.

The objective of this project was to build a system that processes tweets, performs sentiment analysis, and visualizes the results through an interactive dashboard that allows users to quickly explore sentiment trends.

Dataset

The project uses a subset of the Sentiment140 dataset, which contains tweets labeled with sentiment information. A collection of over 500 tweets was used for analysis and visualization within the dashboard.

Tools Used

  • Python
  • Plotly Dash
  • VADER Sentiment Analysis
  • Data Visualization
  • Text Preprocessing

Key Insights

The sentiment distribution revealed a mix of positive and negative opinions within the dataset, demonstrating how social media text can reflect public emotions. Visualizations such as bar charts and pie charts made it easy to identify overall sentiment trends in the tweet collection.

Word cloud visualizations highlighted frequently used terms within different sentiment categories, helping identify common themes expressed by users. The interactive dashboard enables quick exploration of sentiment patterns and provides a clear overview of textual data insights.
